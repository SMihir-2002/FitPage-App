## Development

Make Sure you have [Flutter development environment setup](https://docs.flutter.dev/get-started/install)

```bash
# Install dependencies
flutter pub get
```

```bash
# Pod installation for ios
cd ios && pod install
```

```bash
# Run
flutter run
```
